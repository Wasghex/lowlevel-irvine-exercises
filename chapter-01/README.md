# Low-level Irvine Exercises

Exercícios e pequenos projetos baseados no livro  
**Assembly Language for x86 Processors (7ª Edição)** — Kip R. Irvine.

## Sobre
Este repositório contém exercícios de programação em baixo nível, escritos em:
- Assembly x86 (MASM)
- C++

O objetivo é estudar:
- conceitos de baixo nível
- linguagem assembly
- memória e fluxo de controle
- interação entre C++ e assembly

## Estrutura
Cada exercício é um projeto independente, com seu próprio ponto de entrada (`main`).
